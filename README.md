# Фильмопоиск

### Для запуска проекта выполните следующие шаги:

1. Клонируйте репозиторий
   `git clone https://github.com/pltnkvart/filmopoisk.git`

2. Установите зависимости
   `npm i`

3. Запустите команду
   `npm run dev`

4. Запустите необходимые сервера.
